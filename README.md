# sveltekit-demo
Svelte Kit 构建的 Svelte 项目

## 快速开始

```base
npm init svelte@next my-app
cd my-app
npm install
npm run dev

// or

mkdir my-app
npm init svelte@next
npm install
npm run dev
```
